# -ATM-Simulation-C-
ATM functioning simulated in c++ with enhanced security features.
<b>
Built a C++ Windows ATM simulation with multi-user support, file-based account storage, and secure masked
 PIN login.
 <b>
 Added security features including account lockout after failed attempts and timed auto-unlock.
 <b>
  Implemented core ATM functions: balance check, withdrawals with limits, deposits, and mobile number updates.
  <b>
   Applied modular OOP design for maintainability and easy future enhancements.
